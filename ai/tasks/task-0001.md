## Constraints
1. `internal/testfx/statefs` MUST NOT import any other `internal/` package
   (per M0 "done when" criterion in `implementation-plan.md`).
2. ULID dependency MUST be `github.com/oklog/ulid/v2` (per `design.md` §13).
   Pin a specific released version (no commit SHA, no `latest` pseudo-version).
3. No `panic()` in production paths. (Test helpers may `t.Fatalf` freely.)
4. No `time.Now()` directly in helpers; if you need a clock, accept it as a
   parameter or skip the dependency entirely for M0 (per cross-cutting rule in
   `implementation-plan.md`).
5. Helpers must be safe for `t.Parallel()`-using callers (each `NewWorkspace`
   returns a distinct temp dir; no shared global state).
6. JSON encoding in helpers must use stable key order and trailing newline if
   they ever write JSON (M0 helpers are mostly readers; if you add a writer
   helper, follow this rule).
